**Main App:**
- `src/App.jsx` - Main application orchestrating views and state

### Design System Implementation:

**Colors (from DESIGN.md):**
- Primary: Cosmic Purple (#6B46C1)
- Card positions: Past (Sage Green), Present (Ocean Blue), Future (Twilight Rose)
- Accents: Golden Glow, success/error states
- Backgrounds: Canvas (#FAFAFA), Surface (white)

**Typography:**
- Display font: Playfair Display (headers, card names)
- Body font: Inter (UI text)
- Mono font: JetBrains Mono (code, stamps)

**Animations (Framer Motion):**
- Card entrance: fade + y-slide, 240ms easeOut
- Card flip: rotateY 180deg, 600ms spring
- Hover: scale(1.02), 120ms
- Confetti: particle burst on export success
- Page transitions: AnimatePresence with exit animations

### Features Implemented:
1. PDF drag-and-drop upload with validation (10MB max, magic bytes check)
2. Three-card tarot spread with flip reveal animation
3. Aura badge with gradient styling per type
4. Certification stamp with vintage rubber stamp aesthetic
5. Export button with loading spinner and success confetti
6. Progress indicator during analysis
7. Error handling with retry functionality
8. Responsive design (mobile card stack, desktop spread)
9. Reduced motion support
10. Mock API for development without backend
